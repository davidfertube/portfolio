# PPE Detection v2 (Synthetic)

A high-quality synthetic dataset for Personal Protective Equipment (PPE) detection in industrial environments. 

## Dataset Description
- **Curated by:** David Fernandez
- **Format:** YOLOv8 / COCO
- **Classes:** Hard-hat, Safety-vest, Gloves, Glasses, Boots, No-PPE.

## Use Case
Perfect for training edge-AI models like YOLOv8 or EfficientDet for real-time safety compliance monitoring where real-world annotated data is scarce or sensitive.
